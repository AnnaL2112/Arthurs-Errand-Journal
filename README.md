# Arthur's Errand Journal

A local browser tracker for Red Dead Redemption 2 playthroughs, 100% completion, optional errands, collectibles, challenges, chapter reminders, mysteries, treasure and freeform trail notes.

## Version
v1.0

## Notes
- Opens on the Playthroughs page so users can create, load or switch playthroughs before reaching the dashboard.
- Playthroughs can be tagged Vanilla or Modded.
- Section-level toggles reveal or hide locations/details where needed.
- Trail Journal adds freeform notes for the active playthrough.
- Trail Journal notes support add, edit, delete, search, user-made tags, tag filtering and pinning important notes to the top.
- Trail Journal notes save inside the current playthrough and export/import with the playthrough JSON.
- Camp donation tracker uses add/subtract controls instead of requiring manual total edits.
- Progress is saved locally in the browser. Export your playthrough JSON before clearing browser storage or moving computers.
